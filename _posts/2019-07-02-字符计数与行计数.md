### 字符计数

```c
#include<stdio.h>

int main(){
    long nc;
    nc = 0;
    while(getchar()!=EOF)
        ++nc;
    printf("%ld\n",nc);
        
}
```

![](https://nikolablog-1258612035.cos.ap-shanghai.myqcloud.com/20190702180315.png)

下面是for循环的代码：

```c
#include<stdio.h>

int main(){
    double nc;
    for(nc = 0;getchar()!=EOF;++nc)
        ;
    printf("%.0f\n",nc);
    return 0;
}
```

### 行计数

统计行数等价于统计换行符的个数

```c
#include<stdio.h>

int main(){
	int c,n1;
	n1 = 0;
	while((c=getchar())!=EOF)
		if(c=='\n')
            ++n1;
    printf("%d\n",n1);
    return 0;

}
```

![](https://nikolablog-1258612035.cos.ap-shanghai.myqcloud.com/20190702182850.png)

以上的单引号中的字符表示一个整型值，注意'\n'与"\n"的不同，前者单个字符，是个整型数。后者是仅仅包含一个字符的字符串常量。

### 习题

###### 编写一个统计空格、制表符与换行符的个数的程序

```c
#include<stdio.h>

int main(){
	int c,n1,n2,n3; 
	n1 = 0;
	n2 = 0;
	n3 = 0; 
	while((c=getchar())!=EOF){
	
		if(c==' ')
            ++n1;
        if(c=='\t')
            ++n2;
        if(c=='\n')
            ++n3;
    }
    printf("空格个数:%d\n",n1);
    printf("制表符个数:%d\n",n2);
    printf("换行符个数:%d\n",n3);
    return 0;

}
```
